# Responsi Modul 5 (Coffee Brewer)

**Coffee Brewer** adalah sistem kontrol digital berbasis Verilog yang dirancang untuk diimplementasikan pada FPGA Nexys A7 guna mengotomatisasi proses pembuatan kopi melalui Finite State Machine (FSM) dengan tahapan grind, brew, dan drip , yang dilengkapi indikator keluaran perangkat keras berupa LED dan seven-segment display.
